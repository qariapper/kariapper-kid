Genealogy Numbering Systems

In the Kariapper Family Tree, we use three unique yet interconnected systems to identify and trace members of the lineage. Each serves a specific purpose. Here's a summary:

1. 🧬 dID – d'Aboville ID

What it is:
A dot-separated genealogical path showing a person’s lineage. This follows the standard d’Aboville system, except we start with ALK (the founder) instead of 1.

Example: ALK.1.2.4
→ 4th child of the 2nd child of the 1st child of ALK

2. 🪪 KID – Kariapper ID

What it is:
A structured ID that encodes birth order, gender, and optional attributes like number of spouses or tuplets.

Example: ALK.1x2s.2y2t.3x
→ ALK’s 1st child (female with 2 spouses), her 2nd child (male, part of twin), his 3rd child (female)

3. ✍️ KSID – Kariapper Short ID

What it is:
A compact letter-based ID encoding birth order and gender from ALK down to the person. Each letter = one generation.

- Uppercase = Male
- Lowercase = Female
- A = 1st child, B = 2nd child, ..., Z = 26th child

Example: AAaB
→ ALK’s 1st son → his 1st daughter → her 2nd son
